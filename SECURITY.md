# Security Policy

This is a simulation-only package with no network access, no credential
handling, and no native code beyond its declared dependencies (`mujoco`,
`gymnasium`, `numpy`).

If you believe you've found a security issue, please report it privately via
[GitHub's private vulnerability reporting](https://github.com/isaka1022/so101-sim2real/security/advisories/new)
rather than opening a public issue.
